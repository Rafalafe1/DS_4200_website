About Me & What I Build

I focus on creating small, useful web projects that prioritize clarity, accessibility, and maintainability. Over time I’ve learned to break down larger ideas into simple, shareable components: a readable HTML structure, consistent typography, and modest CSS that emphasizes content rather than flashy effects. I enjoy experimenting with layout systems (flexbox and grid), responsive behavior, and a few carefully chosen interactions that improve usability without adding cognitive load. My projects often start as a personal need usually for finance or class projects. Sharing work publicly on GitHub helps me get feedback, iterate, and learn from other contributors. I also document decisions and trade-offs so the code remains approachable for newcomers. Outside of the browser, I enjoy photography, grilling and reading the Bible.

Below are a few recurring topics and micro-project ideas I enjoy working on. Each is approachable as a short learning exercise and can be extended into a more complete project when needed.

  • Webhook Trader: Allows for trading from tradingview to the IBKR API automatically
  • Photography portfolio: Image grids with lazy loading and descriptive captions for accessibility.
